GitZipQR

Secure archives through QR-coded encryption

GitZipQR turns a folder into a deterministic ZIP, encrypts it with AES-256-GCM and stores the ciphertext inside QR codes.

  1. Zip the folder with normalized timestamps
  2. Derive a key via scrypt and encrypt with AES-256-GCM
  3. Split the ciphertext into QR-sized chunks
  4. Embed each chunk directly into a QR image (base64 payload)
  5. To restore, scan all QR codes and decrypt using the same password
